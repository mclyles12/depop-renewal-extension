How to use Depop Renewer

Automatically keeps your listings fresh in Depop search — no paid subscriptions, no bloat.

1 — First Time Setup

1
Make sure you're logged into Depop in Chrome. The extension uses your active session to save listings.
2
Click the extension icon in your Chrome toolbar to open the popup.
3
In the Scrape field, enter your Depop profile URL — e.g. depop.com/maggiemadethis/ — and click ⟲ Scrape. It will open your profile in a background tab, scroll through all your listings, and import them automatically.
4
If you have listings under multiple shop names, repeat the scrape for each profile URL. Listings are merged automatically.
⚠ During scraping: Background tabs will open and close on their own. Don't click or close them — just leave Chrome alone until it finishes.

2 — Setting a Schedule

1
Pick a refresh interval using the pill buttons: 4h, 8h, 12h, 24h, or 48h. This controls how often your listings are automatically renewed.
2
Toggle the switch in the top-right of the popup to ON. The dot will turn green and pulse to confirm it's running.
3
The Next Run counter shows when the next automatic renewal will happen. It uses a small random offset each time so it never fires at the exact same time.
Tip: The extension runs while Chrome is open. If you close Chrome, the schedule pauses and resumes next time you open it.

3 — Manual Renewal

1
Hit ⟳ Renew All Now at any time to immediately renew all listings, regardless of the schedule.
2
Listings are renewed in reverse order so that listing #1 ends up on top in search when it's done.
3
Watch the Activity Log at the bottom of the popup for live status — green checkmarks mean success, red X means it couldn't find the save button on that listing.

4 — Visual Layout Manager

1
Click the ⊞ grid icon in the top-right of the popup to open the Layout Manager.
2
Drag and drop your listing photos to arrange them in the order you want them to appear in Depop search.
3
Click 💾 Save Order to save your arrangement, or ⟳ Renew in This Order to save and immediately renew so the order takes effect in search right away.
How ordering works: Depop shows recently refreshed listings first. By renewing in reverse order (last listing first, first listing last), listing #1 gets renewed last and ends up at the top of search.

5 — Things to Know

Don't touch background tabs. When renewing, the extension opens each listing's edit page in a background tab and clicks Save. If you interact with those tabs, the save may fail or save incorrect data.
If a listing shows ✗ in the log: Depop may have updated their website layout and changed the Save button. Report it as a bug so it can be fixed.
Depop must be logged in. The extension uses your active Depop session. If you get logged out, renewals will fail silently. Check the Activity Log if things stop working.

FAQ

Does this violate Depop's terms of service?
The extension mimics the same action as manually opening a listing and clicking Save — it doesn't use any unofficial APIs or inject fake data. Human-speed delays are built in to keep behavior natural. Use at your own discretion.
Why did the scraper only grab some of my listings?
Depop uses infinite scroll — listings load as you scroll down. If the scraper stopped early, your profile may have taken longer to load than expected. Try scraping again while your profile page is already open in Chrome.
Will this work if Chrome is closed?
No — the schedule only runs while Chrome is open. The extension will pick up where it left off next time Chrome is open.
How do I add a listing manually?
Go to the listing on Depop, click Edit, copy the URL from the address bar — it should look like depop.com/products/edit/your-item-slug/ — and paste it into the Add field in the popup.
